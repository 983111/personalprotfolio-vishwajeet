KhetiAI

AI-Powered Agriculture Platform

üöÄ Overview

KhetiAI is an AI-powered digital agriculture platform built to help farmers across India manage their operations more efficiently. The platform combines crop disease detection, weather forecasting, budget tracking, and an equipment marketplace into one comprehensive mobile application.

Started in June 2025, I led a team of 7 members from around the world to build a solution that addresses real problems faced by farmers in Maharashtra and Telangana. The app integrates Google's Gemini 2.5 Flash for AI-powered crop disease detection, providing instant analysis and treatment recommendations.

What makes KhetiAI different is that it's built by someone who understands farming. My family runs an agriculture shop, and I've helped farmers apply for government subsidies. This direct experience shaped every feature in the app.

Gemini API Integration

The disease detection system uses a Cloudflare Worker to handle image uploads and communicate with Gemini's API. Here's how we handle retries when the API is rate-limited:

let apiResponse;
for (let attempt = 0; attempt < MAX_RETRIES; attempt++) {
    try {
        const response = await fetch(apiUrl, {
            method: 'POST',
            headers: { 'Content-Type': 'application/json' },
            body: JSON.stringify(payload)
        });

        if (response.ok) {
            apiResponse = await response.json();
            break;
        }

        if (response.status === 429 || response.status >= 500) {
            if (attempt < MAX_RETRIES - 1) {
                const delay = getExponentialBackoffDelay(attempt);
                await new Promise(resolve => setTimeout(resolve, delay));
                continue;
            }
        }
    } catch (networkError) {
        if (attempt < MAX_RETRIES - 1) {
            const delay = getExponentialBackoffDelay(attempt);
            await new Promise(resolve => setTimeout(resolve, delay));
            continue;
        }
        throw networkError;
    }
}

Structured Response Format

We use JSON schema to force the AI to return consistent results every time:

const responseSchema = {
    type: "OBJECT",
    properties: {
        "plant_type": {
            "type": "STRING",
            "description": "Type of plant (e.g., 'Tomato', 'Rose')"
        },
        "disease_name": {
            "type": "STRING",
            "description": "Disease name or 'Healthy'"
        },
        "severity": {
            "type": "STRING",
            "description": "'Low', 'Medium', or 'High'"
        },
        "treatment_steps": {
            "type": "ARRAY",
            "items": { "type": "STRING" }
        }
    },
    required: ["plant_type", "disease_name", "severity", "treatment_steps"]
};
